User Guide Workouts

This is where your workouts are managed. Each workout appears on a workout card, displaying its name, description and a list of links to each day of the workout. To view a specific day of the workout, tap the DAY button corresponding to the day number.

Update Workout:
Tap the round gear button to switch to Edit mode. Tap the Workout’s name to update it. Tap the Workout’s description to update it.

Delete Workout:
Tap the trash can button to delete the workout.

Add Workout:
Tap the rounded plus button (to the right of the top left X button). Edit the name and description.

Export Workout:
Tap the share-social button (below top left X button). Select ‘upload to the cloud’ to store the workout online. You can send the workout id to other users so they can import it. The id is displayed below the share-social button and can be copied by tapping the copy button on the right.

Share Workout:
Tap the share-social button (below top left X button). Select ‘Share via SMS’ to open the SMS app on the device to send your user id and workout id to other users.

Import Workout:
Tap the download button (below top left X button). Enter the user id of the workout’s creator and the workout id. This will download the workout and import it into your workouts list.

View Workout Day:
The workout day displays its name and the list of exercise. Each exercise displays its name, properties and a media image. Tapping the image will take you to the Exercises tab (see more later). Swipe up or down to scroll through all the exercises for that day. Swipe left or right to switch to other days in the workout.

Workout Day actions:
Tap the top right calendar button.
To add a day, tap the plus button below the top right X button.
To delete a day, tap the minus button below the top right X button.

To shift a day backward or forward, tap the left or right buttons below the top right X button correspondingly.

Working out:
Tap the bottom right play button to start working out on the selected day. Each set in the exercise will be horizontally aligned and will display the set’s reps, weight, and time. The current set will animate while it’s active.

When you’ve completed a set, tap the Done! Button to move to the next set. If the exercise has a rest time between set, rest progress time will be displayed.

To skip the rest, tap the skip button.
To skip sets, tap the forward button.
To repeat a set, tap the backward button

To view a larger image of the exercise, tap the image and select ‘View Larger’.
To view more information about the exercise, tap the image and select ‘Find in Exercises’.

At any time, you can switch to any other exercise in the day by tapping the play button at the specific exercise.
To stop the workout before its completed, tap the stop, or the X button on the bottom right.

To mute the set’s cues, tap the mute button. Tap again to unmute.
To show any other exercise’s sets while working out, tap the exercise’s image, and select ‘show set’. To hide it, select ‘Hide set’.

Edit workout Day:
Tap the bottom left gear button to switch into Edit mode.
To update the Day name, tap the it and edit it. While you update the Day’s name, you can also update the number of times to repeat the Day’s exercises.
To update the exercise name, tap it and edit it.

To delete the exercise, tap the image and select ‘Delete this exercise’.
To view more information about the exercise, tap the image and select ‘Find in exercises.

To change the settings of the exercise, tap the settings under the image and update it.
To change a set, tap the set button and select ‘Edit this set’.
To add a set, tap the set button and select ‘Add a copied set’. This will copy the selected set and add it to the end of the sets.
To delete a set, if it’s not the only one, tap the set button and select ‘Delete this set’.

To update the rest times, tap the down arrow to open the rest times and drag any of the ‘Rest between sets’, or ‘Rest after exercise’ bars.

Add an exercise:
Tap the dumbbell button to the right of the bottom left X button. A list of all the exercises in your app will be shown.
Check off any exercise you’d like to add to the current workout day.

Tap the ‘Add Exercise(s)’ button on the bottom left to add the selected exercises and get back to the workout day view.
A ‘Multiset’ is a set with multiple exercises that is executed without rest between the exercises. Check off the multiset option to create Multiset with the selected exercises.
To filter the list of exercises by the muscle groups they use, tap the top left ‘Filter by Muscles’ toggle, and then tap the ‘Filter by Muscles’ button on the right.

Filter exercises by Muscle Group:
Select a muscle group by either tapping on the image’s specific muscle or tapping the muscle group name on the bottom. More than one muscles can be selected. Cardio exercises can also be selected by tapping the heart shape on the image.

Once the muscles are selected, tap the Back button to get back to the exercise list. The list will be filtered by the selected muscles. You can disable the filter by tapping the top left ‘Filter by Muscles’ toggle.

To change the order of the exercises in the Day, drag the ‘reorder handle’ on the middle right of the exercise view up or down to move it back or forth.

Exercises

This is where all of all the exercises in your app are listed. The list can be filtered by muscle group(s) [see above]. Each Exercise has a name and related muscle groups count link.

To change the muscle group associated with an exercise, tap the link button. This will take you to the Muscles selection view where you can select the muscles group for that exercise
To Delete an exercise, tap the trash can button on the right. Note: an exercise cannot be deleted if it is used in a workout.

To view more information about the exercise, tap the chevron button. A description of the exercise is displayed. In addition, a list of ‘walking man’ links will take you to every workout day where this exercise is used.

To change the name of an exercise, tap its name.

To View a larger image of the exercise, tap the image, and select ‘Expanded’.
If an exercise has more than one image, you can change the order of the images by tapping the image and selecting ‘Move Image Forward’ (if it’s not last). The first image in an exercise is the one that appears in the workout day.

To add new exercise image, tap the ‘Add Image’ camera button on the bottom. This will allow you to select an image from your device’s library or to take a new picture using the camera.
To add an image to an existing exercise, tap the exercise’s image, and select ‘Add Image’.
To remove an image from an existing exercise, tap the exercise’s image, and select ‘Remove this Image’.

Settings

Account
This is where your TrainMe account is displayed. If you don’t have an account, you can create one. Once created and verified, you can sign into your account.

Once signed in, your username will be displayed with a sign out button. Your user id will also be displayed below the username. you will need the user id in order to import (download) your exported workouts.

Themes
TrainMe has a variety of themes. To change the theme, select it from the list.

Options
In this segment, you will have links to the User Guide, Release Notes, and Terms of Use & Privacy Policy of TrainMe.

To view the User Guide, tap the link. This user Guide will be displayed. Tap th back button to get back to the Options list.
To view the Release notes, tap the link. A popup will be displayed with a list of various release (version) and their related features.
To view the Terms & Policy, tap the link. Once the document is viewed, you will need to accept the terms if you wish to use all the app’s features.

To restore the app’s data to its initial state, tap the ‘Reset Workout Data’ button.
